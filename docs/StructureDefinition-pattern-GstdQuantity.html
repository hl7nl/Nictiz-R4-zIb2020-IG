Data Type Profile: G-Standard Quantity

Official URL: http://nictiz.nl/fhir/StructureDefinition/pattern-GstdQuantity Version: 0.9.0
Draft as of 2023-10-17 Computable Name: PatternGstdQuantity

The implementer alone is responsible for identifying and obtaining any necessary licenses or authorizations to utilize third party IP in connection with the specification or otherwise.

A Quantity datatype profile to add a Quantity translation in G-Standard units using the HL7 core iso21090-PQ-translation extension. The extension MAY be used to additionally add translations expressed in a different unit from a different unit code system (e.g. NHG table Gebruiksvoorschriften (Table 25)).

This profile provides a uniform way to include the iso21090-PQ-translation extension to provide a Quantity based on G-Standard units.

Usage:

  • Use this DataType Constraint: zib PharmaceuticalProduct
