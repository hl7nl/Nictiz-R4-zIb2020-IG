A derived profile from zib-ContactInformation-TelephoneNumbers to provide a version better suited for implementation purposes. This profile augments the base profile with elements found in the various use cases that have adopted the zib.

To implement the entire zib, the complementary profile nl-core-ContactInformation-EmailAddresses should be used. Both profiles can be added in a slice with the discriminator set to profile/$this.
