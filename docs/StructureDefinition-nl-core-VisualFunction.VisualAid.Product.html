Sight is the ability to observe lit objects, with the goal of orienting yourself on objects and people in a lit environment. This pertains to observing the light stimuli, and not processing them in the brain. A sight disorder can lead to things including orientation problems.

A derived profile from zib-VisualFunction.VisualAid.Product to provide a version better suited for implementation purposes. This profile augments the base profile with elements found in the various use cases that have adopted the zib.

Usage:

  • Refer to this Resource Profile: nl core VisualFunction.VisualAid
  • Examples for this Resource Profile: Device/nl-core-VisualFunction.VisualAid.Product-01
