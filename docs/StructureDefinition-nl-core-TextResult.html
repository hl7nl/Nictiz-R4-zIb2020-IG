Resource Profile: nl core TextResult

Official URL: http://nictiz.nl/fhir/StructureDefinition/nl-core-TextResult Version: 0.9.0

Textual results describe the findings and interpretation of diagnostic or therapeutic procedures carried out on patients or on specimens from the patient.

A derived profile from zib-TextResult to provide a version better suited for implementation purposes. This profile augments the base profile with elements found in the various use cases that have adopted the zib.

Usage:

  • Refer to this Resource Profile: nl core Procedure event
  • Examples for this Resource Profile: DiagnosticReport/nl-core-TextResult-01
