Data Type Profile: zib AddressInformation ( Abstract )

Official URL: http://nictiz.nl/fhir/StructureDefinition/zib-AddressInformation Version: 0.9.0
Draft as of 2023-10-17 Computable Name: ZibAddressInformation

Copyright/Legal: Copyright and related rights waived via CC0, https://creativecommons.org/publicdomain/zero/1.0/. This does not apply to information from third parties, for example a medical terminology system. The implementer alone is responsible for identifying and obtaining any necessary licenses or authorizations to utilize third party IP in connection with the specification or otherwise.

Address information includes data where a person resides (temporarily, permanently) or where a building is located.

This Address datatype represents the Dutch zib (‘Zorginformatiebouwsteen’, i.e. Health and Care Information Model) AddressInformation v1.1 (2020).

Usage:

  • Derived from this DataType Constraint: nl core AddressInformation
  • Use this DataType Constraint: zib ContactPerson, zib HealthProfessional Practitioner, zib HealthcareProvider, zib Patient and zib Payer Organization
