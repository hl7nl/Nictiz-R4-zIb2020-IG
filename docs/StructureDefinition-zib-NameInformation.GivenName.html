Data Type Profile: zib NameInformation.GivenName ( Abstract )

Official URL: http://nictiz.nl/fhir/StructureDefinition/zib-NameInformation.GivenName Version: 0.9.0
Draft as of 2023-10-17 Computable Name: ZibNameInformationGivenName

The name normally used to address the person.

This HumanName datatype represents the GivenName concept of the Dutch zib (‘Zorginformatiebouwsteen’, i.e. Health and Care Information Model) NameInformation v1.1 (2020).

For the entire zib, the complementary profile zib-NameInformation should be used. Both profiles can be added in a slice with the discriminator set to value/use.

Usage:

  • Derived from this DataType Constraint: nl core NameInformation.GivenName
  • Use this DataType Constraint: zib ContactPerson, zib HealthProfessional Practitioner and zib Patient
